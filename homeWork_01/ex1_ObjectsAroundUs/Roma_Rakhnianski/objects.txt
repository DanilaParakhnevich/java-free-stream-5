Примеры объектов:
  1) Объект - Портфель.
     Состояние: 
      открыт/закрыт
      длина ремней
     Поведение:
      открыть - при открытии портфеля появляется возможность положить предметы внутрь
      закрыть - убирается возможность класть предметы
      увеличить/уменьшить длину ремней - позволяет настравивать длину ремней портфеля 
  2) Объект - Стиральная машина.
     Состояние:
      вкл/выкл
      Режим стирки
     Поведение:
      включить - позволяет запустить стиральную машину для дальнейшего использования
      настроить режим - позволяет выбрать предпочтительный для стирки режим
      выключить - позволяет остановить работу объекта
  3) Объект - Кошка.
     Состояние:
      Уровень здоровья
      Уровень комфорта
     Поведение:
      Покормить/напоить - действие, поддерживающее уровень здоровья на приемлемом уровне
      Погладить - действие, поддерживающее уровень комфорта
      При невыполнении данных действий уровни здоровья/комфортно постепенно снижаются
  4) Объект - Часы.
     Состояние:
      текущее время
     Поведение
      Настроить часы - при сбое состояния позволяет вручную настроить данные
  5) Объект - Нож.
     Состояние:
      Уровень остроты
     Поведение:
      Резать - позволяет разделять другой объект на более мелкие части
      Точить - позволяет повысить уровень остроты с помощью объекта "Точильный камень"
  6) Объект - Тарелка.
     Состояние:
      Уровень грязи
     Поведение:
      Положить еду - заполняет данный объект едой
      Почистить тарелку - уменьшает уровень грязи
  7) Объект - Сигарета.
     Состояние:
      Целая/Скуренная
     Поведение:
      Достать сигарету - позволяет использовать данный объект
      Зажечь сигарету - активировать свойства объекта (вдыхание дыма)
      Курить сигарету - при использовании этого поведение объект уменьшается
      p.s. Курение вредит здоровью
   8) Объект - Гитара.
       Состояние:
        Настроенная/Расстроенная
        Кол-во струн
        Длина грифа
       Поведение: 
        Настроить гитару - при использовании этого поведения можно добиться более чистого уровня звука
        Играть на гитаре - производство мелодичного звука при использовании
   9) Объект - Рубашка.
      Состояние:
       Цвет
       Количество пуговиц
       Уровень грязи
      Поведение: 
       Надеть/Снять - прямое назначение объекта для поддержки уровня тепла объекта "Носитель рубашки"
       Пришить пуговицу - при уменьшении числа пуговиц использование данного действия позволяет восстановить исходное значение
       Постирать - при использовании уменьшает уровень грязи
   10) Объект - Яблоко.
       Состояние:
        Уровень зрелости
        Уровень грязи
       Поведение:
        Сорвать яблоко - использование этого действия необходимо для последующих действий с этим объектом
        Помыть яблоко - использование данного действия опционально, однако возможен баг "Больной живот"
        Съесть яблоко - поглощение объекта
